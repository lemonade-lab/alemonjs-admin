# alemonjs web

开发文档 https://lvyjs.dev/

开发文档 [https://alemonjs.com/](https://alemonjs.com/)

## 使用指南

- 加载模块

```sh
yarn add alemonjs-admin
```

`alemon.config.yaml`

```yaml
apps:
  - 'alemonjs-admin'
```

## 开发

```sh
yarn install
```

```sh
# vite
yarn dev
# app
yarn app
```

```sh
yarn build
```
